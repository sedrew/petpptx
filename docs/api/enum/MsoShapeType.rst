``MSO_SHAPE_TYPE``
==================

Specifies the type of a shape

Alias: ``MSO``

Example::

    from pptx.enum.shapes import MSO_SHAPE_TYPE

    assert shape.type == MSO_SHAPE_TYPE.PICTURE
